Definitions

  • HTML(HyperText Markup Language):

    • A collection code or symboles inserted in file which create a website,then the website viewed in Internet.it is statatic webpage content

  • CSS(Cascading Style Sheets):

    • it is designing website page to be attiactive or more presentable in the screen when it is viewed.

  • JavaScript

    • It is a web based programming language that makes the website more interactive and easible to use for user.It performs dynamic pages like animation,display data or responsive to users.

  • HTTP(HyperText Transfer Protocol):

    • HTTP is a protocol used by the World Wide Web,It defines how messages are formatted and transmitted,and what actions Web servers and browsers should take in response to various command.

  • HTML5

    • It is the 5th generation of hypertext marked language which provides,html for structue, cascading styles for presentation and javascript for things to happend.

  • Dynamic HTML

    • it is collection of technologies which helps the website more interactive and animated by using HTMl,Javascript,Css and DOM(Document object Model)

Coding conventions

  • Coding convention are the set of guidelines for a specific programming language that must be follwed when someone is coding
  • It makes the source code easy to readable and maintainable
  • It makes code more readable and manageble,easy to follow the codings

Following are the coding convention i am going to use:

HTML

  • use doctype: <!doctype html>
  • Use lang attribute: <html lang=”en-us”></html>
  • Use soft tabs with two spaces (no tabs key)
  • use elements name: <h2> This is a example </h2>
  • use closeing html tags
  • Use lowercase with quote attribute: <p id=”age” >Age is just a number!</p>
  • Use soft tabs with two spaces
  • Use attribute in order

CSS

  • Include one space after : for each declaration.
  • finishing each declarations with semicolon
  • use Lower case all hex values
  • Avoid specifying units for zero values: margin: 0;
  • specifying order in positioning, box model, typographic, visual.
  • commenting on the code
