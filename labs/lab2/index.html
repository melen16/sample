2. How CSS media queries can help you build responsive websites?

In the past, designers were using resolution aware layouts, which rely on java script driven solution like Cameron Adams excellent script. But the media query specification provides a host of media features that extends far beyond screen resolution, and using much wider scope and we can test multiple property values in a single query by chaining them using and keyword.

Media queries are like conditional statements, it works so that it can shape the layout of our website beyond the initial resolution. By doing so it is not affecting a specific version of browser.

With media queries, we can apply resolution specific spot fixes, adapting our design to better respond to changes in the display.
